# Leinad Macro Bot

This was initially a rewrite of [TheUnifox/LCVRBot](https://github.com/TheUnifox/LCVRBot) to Rust with some additional features and nice to haves.

However modern versions of this bot do more than deal with macros (moderation, event management, etc). These features have been added to a fork of this project which will not be open sourced. Since this repo is no longer being used, it will be archived indefinitely.
